# Wireframes Textuais

## 1. Login
- Logo
- Botão Google OAuth
- Campo e-mail + botão enviar link mágico
- Aviso de consentimento

## 2. Dashboard
- KPIs (reuniões gravadas, horas transcritas, tarefas, decisões)
- Fila de processamento
- Alertas de erro

## 3. Lista de reuniões
- Filtros: data, participante, tipo, tags
- Barra de busca
- Cards com título, data, duração, status

## 4. Página da reunião
- Player de áudio/vídeo
- Transcrição sincronizada (com destaques)
- Resumo estruturado
- Tarefas com responsáveis
- Decisões
- Botões de exportação

## 5. Chat com reunião
- Selector de reunião
- Histórico de mensagens
- Input de pergunta

## 6. Configurações
- Gestão de time
- Retenção de dados
- Integrações

## 7. Admin
- Status de jobs
- Logs de erro
- Auditoria
